#pychat
***

###1.Introduction
pychat is a C/S structure chat tool with friendly UI.It supports

- registration
- login
- add friend
- multi-session
- online/offline message
- online/offline file transmission
- online/offline remind

###2.Dev

- Python2.7
- PyQt4


###3.Deplyment

The server's default IP is 127.0.0.1.
