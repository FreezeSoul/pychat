1.Introduction: OC is a C/S structure chat tool with friendly UI.It supports registration,login,add friend, multi-session, online/offline message,online/offline file transmission, online/offline remind and so on.
2.OC is written in Python2.7 and PyQt4. Its UI is designed by QT designer.
3.The server's default IP is 127.0.0.1.
